Oh, and here are some [examples](https://github.com/shippingfandom/Marinette/tree/main/scripts) of Riddle transpiled to Glosure scripts that you can run with [5hell](https://github.com/jhook777/5hell-for-Grey-Hack-the-Game)/[Marinette](https://github.com/shippingfandom/Marinette)! 🥰
